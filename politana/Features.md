# Feature list

* Fast, signal-based reactivity
* Nested state objects
* Type-safe HTML, CSS, and DOM bindings
* Create managed objects and closures scoped to components
* As-needed and deduplicated resource fetching
* Zero-config site packaging and optimization
* Zero-config static site generation
* Built-in routing
* Built-in reactive WebGPU
* Built-in debug web server
* Custom-built state, DOM, and navigation inspection over WASM
* Interoperability
    * Embed your own HTML, CSS, and JS
    * Direct DOM access for custom components
* Automatic HTML escaping
* Unsafe- and unwrap-free
    * Errors are reported immediately and stop your app
