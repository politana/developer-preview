# Politana Developer Preview

This is experimental software! It has bugs and the API is expected to change.

To try it out, start by installing Rust (and Rust Analyzer for VS Code) on your computer. Then open the `politana` folder in VS Code, navigate to `politana/examples`, open one of the example files, and press the "Run" button above the main function. This will spawn a local server where you can interact with the generated website.
